BDS Automation - Utility Company Workflow

A sophisticated n8n automation workflow designed for Severn Trent Water to process Big Difference Scheme (BDS) applications automatically. The system handles customer data validation, discount calculations based on income thresholds, database operations, and comprehensive audit logging. Features advanced error handling, real-time notifications, and seamless integration with existing utility company systems.

BDS Workflow Architecture

Technology Stack

  • n8n Workflow Engine
  • JavaScript/Node.js
  • CSV Data Processing
  • Auriga SFTP Integration
  • PostgreSQL Database
  • Email Notifications
  • Slack Integration
  • Gmail API
  • Error Monitoring
  • Audit Logging

Workflow Overview

The BDS automation workflow processes customer applications for water bill discounts based on household income. The system automatically validates customer data, calculates appropriate discount percentages, handles database operations, and provides comprehensive reporting and audit trails.

Key Components

Data Ingestion & Validation

Automatically retrieves customer data from Auriga systems via SFTP, parses CSV files, and performs comprehensive validation including income thresholds, account number format checks, and data completeness verification.

Discount Calculation Engine

Implements sophisticated business logic to calculate discount percentages based on household income:

  • Income ≤ £6,000: 90% discount (Extreme poverty)
  • Income ≤ £10,000: 80% discount (Very low income)
  • Income ≤ £15,000: 70% discount (Low income)
  • Income ≤ £18,000: 50% discount (Moderate low income)
  • Income ≤ £21,000: 40% discount (Below average)
  • Income ≤ £23,492: 30% discount (BDS threshold)
  • Minimum payment: £3.90 per month

Database Operations

Handles both successful customer record insertion and error logging with separate database tables for valid customers and processing errors. Includes transaction management and rollback capabilities.

Notification System

Comprehensive notification system including email summaries, Slack alerts, and high-error-rate warnings. Provides detailed processing metrics and business impact analysis.

Key Features

  • Automated CSV parsing and data validation from Auriga systems
  • Dynamic discount calculation based on household income levels
  • Comprehensive error handling and validation reporting
  • Real-time Slack notifications for processing status
  • Email summaries with detailed processing metrics
  • High error rate alerting system with immediate escalation
  • Complete audit trail with confirmation references
  • Database transaction management and error recovery
  • Priority processing for extremely low-income customers
  • Annual savings calculations and business impact reporting

Business Impact

The BDS automation workflow has significantly improved processing efficiency for Severn Trent Water:

  • Reduced processing time from days to minutes
  • Eliminated manual data entry errors
  • Provided real-time processing status and error detection
  • Generated comprehensive audit trails for compliance
  • Enabled automatic prioritization of vulnerable customers
  • Delivered detailed business impact metrics and savings calculations

Error Handling & Monitoring

The system includes sophisticated error handling with multiple levels of monitoring and alerting. High error rates trigger immediate notifications to management teams, while detailed error logs provide actionable insights for data quality improvement. The workflow includes automatic retry mechanisms and graceful degradation to ensure maximum system reliability.
