PropWise - AI Property Intelligence

Comprehensive property analysis platform that leverages AI vision technology to generate detailed reports from satellite and street-level imagery. Features multi-view property analysis, bilingual support (English/Hungarian), interactive controls for image manipulation, and editable AI-generated reports. Integrates multiple Google Maps APIs with GPT-4 Vision for professional property assessment, providing real estate professionals with actionable insights for investment decisions and market analysis.

Project Screenshot

Technology Stack

  • Next.js 14
  • TypeScript
  • TailwindCSS
  • Shadcn UI
  • Google Maps APIs
  • OpenAI GPT-4 Vision
  • Neon PostgreSQL
  • Prisma ORM
  • Vercel

Key Features

  • AI-powered property analysis using GPT-4 Vision technology
  • Multi-view imagery processing (satellite, street-level, aerial)
  • Bilingual support for English and Hungarian markets
  • Interactive image controls and manipulation tools
  • Editable AI-generated property assessment reports
  • Google Maps API integration for location intelligence
  • Professional property valuation and market insights
  • Comprehensive property feature analysis and scoring
