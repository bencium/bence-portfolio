Anna India - Psycho-Oncology RAG Chatbot

A sophisticated conversational AI assistant named "Anna" designed to provide psychological support for cancer patients and their families. Built with Next.js, the system features a comprehensive RAG (Retrieval Augmented Generation) pipeline using Pinecone vector database and integrates with Sarvam.ai's language model for multi-language support across 22 Indian languages. The system includes strict content filtering to avoid medical advice, LangSmith integration for observability, and rate limiting with fallback to Anthropic Claude when needed.

Project Screenshot

Technology Stack

  • Next.js 15.3.3
  • TypeScript
  • React 19
  • TailwindCSS
  • Pinecone Vector DB
  • Sarvam AI API
  • LangSmith
  • Axios
  • React Query
  • Framer Motion

Key Features

  • Multi-language RAG system supporting 22 Indian languages
  • Psychological support focused on cancer patient care
  • Pinecone vector database for semantic search and context retrieval
  • Sarvam AI integration with Claude fallback for reliability
  • LangSmith observability for conversation monitoring
  • Strict content filtering to avoid medical advice
  • Rate limiting and error handling for production stability
  • Culturally sensitive responses for Indian healthcare context
